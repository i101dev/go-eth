# Golang & Ethereum

### Golang playground for Ethereum development

-   Hardhat required

---

<strong>Step 1: Open terminal</strong>

```
cd hardhat && npx hardhat node
```

or

```
make hardhat
```

<strong>Step 2: Open separate terminal</strong>

```
go run .
```

or

```
make run
```

---

### API

```
make api
```
